# ng4demo
Angular 4 demo

-- Updated with Angular 4.1.2 & Bootstrap 4

```
npm i -g @angular/cli
ng new ng4demo
ng serve --port 8001
ng build --base-href /apm/

// Universal
npm i universal-cli -g
ung new ungdemo
ung serve --port 8002
``` 

## Tips

```
npm install -g npm-check-updates
ncu -u // update package.json packages only
ncu -a // update dependency in package.json file
```
